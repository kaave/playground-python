# My python playground

## System requirements

* `>= Python 3.6`

## initialize

```bash
# init local environments
python3 -m venv .venv
# enable local environments
. .venv/bin/activate
# upgrate pip
pip install --upgrade pip
# install all pip modules
pip install -r requirements.txt
```

## How to use 

```bash
# run REPL
ipython
# add pip library
pip install XXX
# ...and export pip modules list
pip freeze > requirements.txt
```
